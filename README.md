# ML Comeback Projects: Heart Disease & ECG Classification (Day 1–3)  
Practical machine learning projects covering heart disease prediction, ECG signal classification, feature analysis, and model evaluation using Python, Scikit-learn, and data visualization.


### Day 1: ECG Signal Exploration
- Plotted ECG signals, added noise, and built preprocessing environment for ML experiments.

### Day 2: Heart Disease Classification
- Built baseline models, compared Logistic Regression, Random Forest and Decision Tree, scaled features, and analyzed feature importance.

### Day 3: ECG Classification
- Preprocessed ECG data, extracted features, built and evaluated ML models (Logistic Regression, Random Forest), and analyzed class-wise performance.


**Key Skills Demonstrated:**  
Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook, data preprocessing, feature engineering, ML model evaluation.
